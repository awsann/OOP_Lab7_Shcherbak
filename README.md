# Lab-7: Serialization / Deserialization (SERIALIZATION/DESERIALIZATION. SAVING AND LOADING OF OBJECTS)
## Мета роботи
Модернізація програми попередніх лабораторних робіт і реалізація механізмів збереження та відновлення об'єктів предметної області:
- Серіалізація та десеріалізація колекції `List<T>` у формати **CSV** та **JSON**.
- Забезпечення коректної обробки пошкоджених/неповних рядків при CSV-десеріалізації.
- Серіалізація/десеріалізація всієї колекції для JSON з контролем імен властивостей та вибірковою серіалізацією властивостей.
- Розширення меню програми для роботи з файлами.
- Додавання/коригування unit-тестів і досягнення їхнього успішного проходження.
## Опис програми
Консольний додаток керує колекцією об'єктів `Film` через клас `Program`. Основні можливості:

- Клас `Film`:
  - Властивості: `Title`, `Director`, `ReleaseYear`, `Genre`, `Rating`, `Id`.
  - Валідація вхідних даних.
  - Методи `IsClassic()`, `IsHighRated()`, `IsModern()`.
  - Статичні методи `Parse()` і `TryParse()` для створення з рядка.
  - `ToString()` повертає рядок з обраним роздільником (`,` або `;`) для CSV.
  - Атрибути/налаштування для JSON-серіалізації (перейменування полів, включення тільки основних властивостей).

- Серіалізація/десеріалізація:
  - **CSV**:
    - Файл містить заголовок.
    - При зчитуванні додаються тільки коректно десеріалізовані об'єкти; пошкоджені рядки пропускаються.
    - Після десеріалізації виводиться кількість доданих об'єктів.
  - **JSON**:
    - Збереження та відновлення **всiєї** колекції як масив JSON-об'єктів.
    - Під час серіалізації властивості можуть бути перейменовані (наприклад `ReleaseYear` → `Year`, `Rating` → `Score`) і серіалізується тільки набір головних полів.
    - Десеріалізація додає всі об'єкти з файлу до колекції (якщо колекція непуста — нові об'єкти додаються).

- Меню програми (розширене):
  1. Додати об'єкт  
  2. Переглянути всі об'єкти  
  3. Знайти об'єкт  
  4. Продемонструвати поведінку  
  5. Видалити об'єкт  
  6. Продемонструвати static-методи  
  7. Зберегти колекцію об'єктів у файлі  
     - 1) зберегти у файл *.csv  
     - 2) зберегти у файл *.json  
  8. Зчитати колекцію об'єктів з файлу  
     - 1) зчитати з файлу *.csv  
     - 2) зчитати з файлу *.json  
  9. Очистити колекцію об'єктів  
  10. Вийти з програми
